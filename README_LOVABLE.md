## ⚡ Como Usar

### Opção A: RÁPIDO (Recomendado para iniciar)
1. Abra **LOVABLE_PROMPTS_CURTOS.md**
2. Copie o prompt do Prompt 1 (Login)
3. Cole no Lovable
4. Customize nomes (Dicoonseguros, textos português)
5. Cole a **instrução de consistência** ao final
6. Gere! ✨
7. Repita para Prompts 2-17 em ordem

### Opção B: DETALHADO (Para refino premium)
1. Abra **LOVABLE_PROMPTS.md**
2. Leia o Design System completo first
3. Copie cada prompt (já com instruções de UX)
4. Cole no Lovable
5. Ajuste conforme necessário
6. Mantenha consistência visual

### Opção C: PLANEJAR ANTES (Metodologia)
1. Abra **LOVABLE_PRD.md**
2. Leia "Ordem de Implementação Recomendada"
3. Escolha MVP (Fase 1) ou implementação completa
4. Marque qual versão de prompt vai usar (A ou B)
5. Depois escolha Opção A ou B acima

## 🎨 Design System (Resumido)

```
Primária:    Azul Petróleo / Teal (#0D5E5C ou similar)
Sucesso:     Verde (#22C55E)
Alerta:      Âmbar (#FFA500)
Erro:        Vermelho (#EF4444) - apenas crítico
Neutros:     Cinzas (backgrounds claros em light, escuros em dark)

Navegação:   Sidebar + Header (desktop) | Bottom Nav (mobile)
CTA:         "Nova cotação" sempre visível
Componentes: Cards, Tabelas, Badges, Avatares, Forms em wizard
Dados:       Mockados REALISTAS
```

## 💡 Exemplo: Colando Prompt 1 (Login) no Lovable

**Passo 1:** Acesse seu workspace Lovable

**Passo 2:** Crie novo projeto (ou use existente)

**Passo 3:** Cole este texto:

```
Crie tela de login premium para plataforma de multicálculo de seguros.

Layout split:
- Esquerda: Logo, headline "Centralize cotações, propostas e conversões em uma única plataforma.", 3 benefícios
- Direita: Card com campos e-mail, senha, botão "Entrar", "Esqueci a senha", "Teste grátis"

Estilo: Premium, corporativo, ilustração abstrata sutil de dados, excelente hierarquia.
UX: Autofoco e-mail, validação clara, mobile impecável, Dark/light mode.

Mantenha o mesmo design system em todas as telas:
- Tipografia sofisticada e corporativo (nada ERP antigo)
- Paleta: azul petróleo/teal primária | verde sucesso | âmbar alerta | vermelho erro apenas
- Sidebar elegante + header superior com busca, notificações, perfil
- Cards discretos com sombra sutil
- Tabelas modernas com bom contraste
- Badges elegantes por status
- Light e dark mode consistentes
- Componentes reutilizáveis (buttons, inputs, forms)
- Dados mockados muito realistas
- CTA "Nova cotação" sempre visível quando faz sentido
- Design premium, corporativo, moderno, zero visual genérico
```

**Passo 4:** Clique "Generate" (ou equivalente no seu Lovable)

**Passo 5:** Review, ajuste, aprove! ✅

## ❓ FAQ

**P: Posso pular etapas?**  
R: Sim, mas mantenha a ordem dos primeiros 5 (Fase 1) para consistência.

**P: Devo usar Curtos ou Detalhados?**  
R: Comece com CURTOS se quer rapidez. Use DETALHADOS se quer premium.

**P: Quanto tempo leva?**  
R: MVP (5 telas) = 1-2 sem. Completo (17 telas) = 1-2 meses.

**P: Posso customizar depois?**  
R: Sim! Os prompts são base. Customize texto, cores, campos conforme negócio.

**P: As telas funcionam offline?**  
R: Depende do Lovable. Geralmente requer conexão para APIs.

**P: Preciso de backend?**  
R: Esses prompts criam frontend. Backend você integra depois (APIs).
