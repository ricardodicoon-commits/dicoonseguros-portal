**3. Usuários e Permissões:** tabela usuários (nome, email, role, status) + botão "+ Novo". Modal: nome, email, role dropdown (admin/gestor/corretor/operacional/leitor), departamento, permissões checkboxes. Ações: editar, desativar, resetar senha.

**6. Notificações:** toggles e-mail, toggles in-app, connect SMS/WhatsApp provider, setup webhook, teste.

**7. Templates Proposta:** cards templates (preview) + botão duplicar. Editor visual drag-drop com campos dinâmicos {{cliente}}, {{valor}}, {{seguradora}}.

**Estilo:** Área admin premium, organizada por abas, clara e profissional.

**UX:** Abas sticky no topo, save automático, feedback "Salvo com sucesso", confirmação ao deletar, preview para identidade visual, teste conexão.

## 📌 INSTRUÇÃO DE CONSISTÊNCIA (Copie e Cole Ao Final de CADA Prompt)

```
Mantenha o mesmo design system em todas as telas:
- Tipografia sofisticada e corporativa (nada ERP antigo)
- Paleta: azul petróleo/teal primária | verde sucesso | âmbar alerta | vermelho erro apenas
- Sidebar elegante + header superior com busca, notificações, perfil
- Cards discretos com sombra sutil
- Tabelas modernas com bom contraste
- Badges elegantes por status
- Light e dark mode consistentes
- Componentes reutilizáveis (buttons, inputs, forms)
- Dados mockados muito realistas
- CTA "Nova cotação" sempre visível quando faz sentido
- Design premium, corporativo, moderno, zero visual genérico
```
